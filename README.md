# rock-paper-scissors

A simple game of Rock - Paper - Scissors. 
The first stage of the development was to make a console game. 
The aim was to create the logic of the game with JavaScript.

At this stage, the game has a UI. 
The aim is to arrange and change the code so that it is based on DOM Manipulation.
The development of the new version took place in a new git branch that was merged to main after the version's completion.
